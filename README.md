# Professional Portfolio - Responsive Web Layout 🚀

## 📖 Project Description
This is a fully responsive, professional personal portfolio built as part of the DecodeLabs industrial training program. The core objective of this project is to demonstrate advanced frontend architecture, ensuring seamless adaptation across mobile, tablet, and desktop screens without relying on fixed dimensions.

**Key Features:**
* **Mobile-First Approach:** Engineered fluidly from mobile to desktop.
* **Advanced Layouts:** Utilized **CSS Grid** for macro-architecture and **Flexbox** for micro-components.
* **Fluid Typography:** Implemented `clamp()` for dynamic font scaling.
* **No-JS Navigation:** Built a fully functional mobile popover menu and content filtering system using pure CSS (`:target` pseudo-class).

## 🛠️ Tech Stack
* HTML5
* CSS3 (Media Queries, Flexbox, Grid)

## 🚀 How to Run the Project
1. Download or clone this repository to your local machine.
2. Navigate to the project folder.
3. Simply double-click the `index.html` file to open it in any modern web browser (Chrome, Edge, Firefox).
4. No installation or local server is strictly required, though you can use VS Code Live Server for hot-reloading.
